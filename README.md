# AI+具身智能资讯聚合

每日自动更新的 AI 与具身智能领域资讯聚合网站，覆盖算法突破、企业动态、政策信号、市场数据等多维度信息。

## 内容板块

- **热点聚焦**：跨类别含金量最高的每日热点
- **创业板块**：具身智能赛道创业公司动态与择业价值判断
- **产品经理视角**：场景判断、标准化机会、产品形态演进
- **算法工程师视角**：技术路线分析、开源模型选型、算力方向
- **项目经理视角**：项目节奏、风险清单、跨团队协作
- **行动建议**：技术选型、赛道判断、关注清单

## 访问方式

GitHub Pages：`https://<你的用户名>.github.io/ai_news/`

## 数据更新

- 自动化每日 18:00 更新
- 数据文件归档于 `data/YYYY-MM-DD.json`
- 网站入口：`site/index.html`

## 本地开发

```bash
node build.js   # 根据 data/*.json 重新生成 site/index.html
```
